## TIPS

- All photos show with a dark green overlay — slightly underexposed/moody photos
  with clear subjects work best. Avoid very bright, washed-out images.
- Name files with hyphens, no spaces: my-photo.jpg not my photo.jpg
- Compress before uploading: use squoosh.app (free) to reduce file size
  without losing quality. Target under 300KB per image.
- Alt text is already written in the HTML for SEO — no need to change it.
- Images that fail to load gracefully show a fallback placeholder,
  so the site won't break if a file is missing.
